# Hostel Complaint Management System
A modern Hostel Complaint Management System developed using Flask, SQLite, HTML, CSS, and Bootstrap.

## Features
* User Registration & Login
* Admin Panel
* Complaint Management
* Complaint Status Tracking
* Responsive Dashboard
* Modern UI Design
* SQLite Database Integration

## Tech Stack
* Python
* Flask
* SQLite
* HTML
* CSS
* Bootstrap

## Live Demo
https://hostel-complaint-system-yyy6.onrender.com

## Installation
git clone https://github.com/atiktadvi/hostel-complaint-system.git
cd hostel-complaint-system
pip install -r requirements.txt
python app.py
